# GREEN.GG
>Green.gg는 당신이 롤을 플레이 한 만큼 붉은 잔디가 심어지는 페이지입니다.
> 
# 주요 기능
> - ###판수에 따른 색상 지정
> > - 12판 초과 "#FF0000"<br>
> > - 9판 초과 "#FF4000"<br>
> > - 6판 초과 "#FF8000"<br>
> > - 3판 초과 "#FFBF00"<br>
> > - 0판 초과 "#FFFF00"<br>
> - ###사용법
> > - 메인 페이지에서 소환사 이름을 검색합니다.
> > - 마우스를 올릴시, 몇월 며칠, 몇판의 플레이를 하였는지 확인 가능합니다.
> 

# 패치 예정
> - ###디자인 작업
> > - 메인 페이지 디자인
> > - green 페이지 디자인
> > - 
> - ###승패 기록
> > - 마우스를 올렸을때 판수를 비롯, 승리 횟수와 패배횟수가 표시됩니다.<br>
